# HCI_Project

可以參考這個影片建環境:https://www.youtube.com/watch?v=Dt3B7W-dFi4 (但最後網站顯示的內容不太一樣)

還需要另外下載`expo-image-picker`跟`lottie-react-native`

執行
```
npx expo start
```

有動畫的那個介面目前只能在手機跑，網站跑會顯示問題

如果要跑llm的部分，需要自己設定一個`key.py`檔案，並且放置OPENAI_API_KEY。
